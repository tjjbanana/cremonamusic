Firebase Setup
Follow these steps once to connect your site to Firebase. It's free and takes about 5 minutes.

Step-by-step Setup

1
Go to console.firebase.google.com and click "Create a project". Give it a name like cremona-site. You can disable Google Analytics if you don't need it.
2
Inside your project, click the web icon </> to register a web app. Give it a nickname (e.g. cremona-web). Copy the firebaseConfig object that appears.
3
Open both admin.html and index2.html in a text editor. Search for PASTE_YOUR_apiKey_HERE and replace the entire config block with your copied config. Do this in both files.
4
In the Firebase console left sidebar, click Build → Realtime Database. Click "Create Database", choose a region, and start in test mode (you can secure it later). Copy the database URL — it looks like https://your-project-default-rtdb.firebaseio.com. Paste it as the databaseURL value in both HTML files.
5
Upload both files to GitHub and enable GitHub Pages. That's it! Your admin panel will now save to Firebase and the website will read from it — visible to everyone, instantly.
6
Security (optional but recommended): In Firebase Console → Realtime Database → Rules, change the rules so only reads are public and writes require auth — or keep test mode for now and secure it later. The admin password in this panel is a simple shared password (not Firebase Auth), which is fine for a small team.
